Stake DAO Staking v2

Sidecar

Git Source

Inherits: ISidecar

Author: Stake DAO

Sidecar is an abstract base contract for protocol-specific yield sources that complement the main locker strategy. It enables yield diversification beyond the main protocol locker (e.g., Convex alongside veCRV) and provides a protocol-agnostic base that can be extended for any yield source. Sidecars are managed by the Strategy for unified deposit/withdraw/harvest operations, with rewards flowing through the Accountant for consistent distribution.

Notes:

  • github: @stake-dao

  • contact: contact@stakedao.org

Registry used to verify the authorized strategy for this protocol

_initialized

Prevents double initialization in factory deployment pattern
