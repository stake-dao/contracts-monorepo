vote

Called by users to vote for pools. Votes distributed proportionally based on weights. Can only vote or deposit into a managed NFT once per epoch. Can only vote for gauges that have not been killed.

Weights are distributed proportional to the sum of the weights in the array. Throws if length of _poolVote and _weights do not match.

function vote(address _ve, uint256 _tokenId, uint160[] calldata _poolVote, uint256[] calldata _weights) external;

Parameters

NameTypeDescription
_veaddressAddress of ve token.
_tokenIduint256Id of veNFT you are voting with.
_poolVoteuint160[]Array of pool ids you are voting for.
_weightsuint256[]Weights of pools.

depositManaged

Called by users to deposit into a managed NFT. Can only vote or deposit into a managed NFT once per epoch. Note that NFTs deposited into a managed NFT will be re-locked to the maximum lock time on withdrawal.

Throws if not approved or owner. Throws if managed NFT is inactive. Throws if depositing within privileged window (one hour prior to epoch flip).

function depositManaged(address _ve, uint256 _tokenId, uint256 _mTokenId) external;

withdrawManaged

Called by users to withdraw from a managed NFT. Cannot do it in the same epoch that you deposited into a managed NFT. Can vote or deposit into a managed NFT again after withdrawing. Note that the NFT withdrawn is re-locked to the maximum lock time.
