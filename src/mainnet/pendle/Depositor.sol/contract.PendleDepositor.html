Stake DAO Lockers

PendleDepositor

Git Source

Inherits: BaseDepositor, SafeModule

Author: StakeDAO

Contract responsible for managing PENDLE token deposits, locking them in the Locker, and minting sdPENDLE tokens in return.

Note: contact: contact@stakedao.org

State Variables

VE_PENDLE

--- CONSTANTS

Address of the vePENDLE token.

address public constant VE_PENDLE = PendleProtocol.VEPENDLE;

Functions

constructor

--- CONSTRUCTOR

Initializes the Depositor contract with required dependencies

If locker and gateway are the same, internal calls will be done directly on the target from the gateway. Otherwise, the gateway will pass the execution to the locker to call the target contracts.

Note: throws: InvalidGateway if the provided gateway is a zero address

constructor(address _token, address _locker, address _minter, address _gauge, address _gateway)
    BaseDepositor(_token, _locker, _minter, _gauge, 104 weeks)
    SafeModule(_gateway);
