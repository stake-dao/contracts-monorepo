Stake DAO Staking v2

CurveFactoryL2

Git Source

Inherits: Factory, Ownable2Step

Author: Stake DAO

CurveFactoryL2 is a specialized factory for deploying Curve vaults on Layer 2 networks. Key differences from mainnet CurveFactory:

  • Integrates with Convex L2 Booster for sidecar deployment
  • Uses ChildLiquidityGaugeFactory array to validate gauges

Notes:

  • github: @stake-dao

  • contact: contact@stakedao.org
