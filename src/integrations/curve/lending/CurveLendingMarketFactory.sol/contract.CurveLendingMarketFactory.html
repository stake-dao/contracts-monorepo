Stake DAO Staking v2

CurveLendingMarketFactory

Git Source

Inherits: Ownable2Step

Author: Stake DAO

Creates a lending market for Curve-associated Stake DAO reward vaults on the given lending protocol

Note: contact: contact@stakedao.org

State Variables

PROTOCOL_CONTROLLER

The address of the Stake DAO Staking v2 protocol controller Used to check if the reward vault is genuine

IProtocolController private immutable PROTOCOL_CONTROLLER;

Functions

constructor

constructor(address _protocolController) Ownable(msg.sender);

deploy

Creates a Stake DAO market on Curve with the specified oracle type

The lending factory must be trusted! The ownership of this contract is propagated to the lending factory

function deploy(
    IStrategyWrapper collateral,
    address curvePool,
    OracleType oracleType,
    OracleParams calldata oracleParams,
    MarketParams calldata marketParams,
    ILendingFactory lendingFactory
) external onlyOwner returns (IStrategyWrapper, IOracle, bytes memory data);

Parameters

NameTypeDescription
collateralIStrategyWrapperThe collateral to use for the market
curvePooladdressThe Curve pool to use for the market
oracleTypeOracleTypeThe type of oracle to deploy (STABLESWAP =1 or CRYPTOSWAP =2)
oracleParamsOracleParamsThe parameters for the oracle (structure works for both types)
marketParamsMarketParamsThe parameters for the market
lendingFactoryILendingFactoryThe factory to use for the market

MarketParams

The parameters needed for the market creation Both the IRM and the LLTV must be valid values pre-approved by the lending protocol.

struct MarketParams {
    address irm;
    uint256 lltv;
    uint256 initialSupply;
}
